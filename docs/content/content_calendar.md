## 📝 Article Template

### Structure for Each Article:

```markdown
# [Article Title]

## Meta Information
- Target Keyword: [primary keyword]
- Secondary Keywords: [2-3 related keywords]
- Meta Title: [max 60 chars]
- Meta Description: [max 155 chars, include CTA]
- Word Count Target: 1500-2500 words
- Target Audience: [engineers / procurement / plant managers]

## Content Outline
1. Introduction (why this matters)
2. Core explanation (what, how, why)
3. Comparison / data table
4. Application examples
5. KARPOL solution (subtle product tie-in)
6. FAQ section (3-5 questions for schema)
7. CTA: Request consultation / download guide

## Internal Links
- Link to: [related product page]
- Link to: [related industry page]
- Link to: [related article]

## Required Media
- [ ] Hero image (1200×675px)
- [ ] Infographic / diagram
- [ ] Product photo (contextual)
- [ ] Technical comparison table
```
