# KARPOL — Global Industrial Manufacturing Website

> Precision Engineered. Globally Delivered.

---

## 🏭 About

KARPOL is a premium industrial manufacturing website for a company specializing in polyurethane, rubber, silicone, Viton, PTFE, aluminum, and engineering plastic components for global industries including marble/stone processing, mining, construction, and industrial automation.

## 📁 Project Structure

```
karpolnet/
├── docs/                    # All project documentation
│   ├── strategy/            # Website strategy & architecture docs
│   ├── development/         # Development guides, changelogs, standards
│   ├── design/              # Visual design system, color, typography
│   ├── seo/                 # SEO strategy, keywords, technical SEO
│   ├── content/             # Content calendar, article drafts
│   └── notes/               # Session notes, decisions, ideas
├── src/                     # Source code (HTML, CSS, JS)
├── public/                  # Static assets
│   ├── images/              # Product photos, factory images
│   ├── fonts/               # Web fonts (Inter, JetBrains Mono)
│   └── assets/              # PDFs, catalogs, downloadable files
└── README.md                # This file
```

## 🎨 Design System

| Element | Value |
|---|---|
| Primary Dark | `#1A1A2E` (Graphite Black) |
| Secondary | `#2D2D44` (Steel Grey) |
| Accent | `#E8611A` (Industrial Orange) |
| Background | `#F5F5F0` (Warm White) |
| Body Font | Inter |
| Mono Font | JetBrains Mono |
